# 定时器管理专精提示词

> 约束 ID：C16.1–C16.3 → [constraint_detail.md](../references/constraint_detail.md)

## 适用场景

FreeRTOS 软件定时器（`xTimerCreate`）：心跳、重连退避、超时检测、周期采样。

---

## 铁律

### C16.1 — 软件定时器回调禁止阻塞

```c
/* ❌ timer 回调中阻塞（timer daemon 任务被卡死，其他 timer 全停） */
static void reconnect_timer_cb(TimerHandle_t xTimer) {
    tls_handshake(host, port, 5000);  /* 阻塞 5s — 所有 timer 冻结 */
}

/* ✅ timer 回调仅发事件，阻塞操作由专用任务执行 */
static void reconnect_timer_cb(TimerHandle_t xTimer) {
    wss_event_t evt = WSS_EVT_RECONNECT_TIMER;
    xQueueSend(s_wss_event_queue, &evt, 0);
}
```

**原因：** FreeRTOS 软件定时器共享一个 daemon 任务，回调阻塞 = 所有定时器停止。

### C16.2 — 动态创建的 timer 须有 delete 路径

```c
/* ✅ timer 生命周期管理 */
static TimerHandle_t s_heartbeat_timer = NULL;

void heartbeat_start(uint32_t interval_ms) {
    if (s_heartbeat_timer == NULL) {
        s_heartbeat_timer = xTimerCreate(
            "hb", pdMS_TO_TICKS(interval_ms),
            pdTRUE, NULL, heartbeat_timer_cb);
    }
    configASSERT(s_heartbeat_timer != NULL);
    xTimerStart(s_heartbeat_timer, 0);
}

void heartbeat_stop(void) {
    if (s_heartbeat_timer != NULL) {
        xTimerStop(s_heartbeat_timer, pdMS_TO_TICKS(100));
        xTimerDelete(s_heartbeat_timer, pdMS_TO_TICKS(100));
        s_heartbeat_timer = NULL;
    }
}
```

**禁止**创建后无 delete 路径（嵌入式 timer 句柄 = RAM）。

### C16.3 — 周期 vs 单次定时器

```c
/* 周期定时器：自动重载 */
TimerHandle_t hb = xTimerCreate("hb", pdMS_TO_TICKS(30000),
                                 pdTRUE,  /* auto-reload */
                                 NULL, heartbeat_cb);

/* 单次定时器：触发一次后停止 */
TimerHandle_t timeout = xTimerCreate("tout", pdMS_TO_TICKS(5000),
                                      pdFALSE,  /* one-shot */
                                      NULL, timeout_cb);
/* 重新使用需手动 xTimerReset */
xTimerReset(timeout, 0);
```

## Agent 检查清单

- [ ] timer 回调是否无阻塞操作（无网络 IO / mutex / delay）？
- [ ] 动态创建的 timer 是否有对应的 stop + delete？
- [ ] 单次定时器是否正确使用 `pdFALSE` + `xTimerReset`？
- [ ] timer 名称是否可读（便于调试 `xTimerGetName`）？
