# 音视频编解码 / 媒体格式一致性专精提示词

适用：I2S/PCM/AEC/ASR/Opus/AAC、Camera/LCD/视频编码、RGB/YUV 像素格式、stride、frame duration、sample rate、bit depth、channels、codec 生命周期。

## C26.1 — 音频格式端到端一致（P0）

- I2S、AEC、ASR、encoder、uplink 必须明确 sample rate、channels、bit depth、endianness。
- 任意两段格式不同，必须有显式 resampler / channel mixer / bit depth converter，并标注输出格式。
- 禁止静默把 48 kHz PCM 当 16 kHz 送 ASR，或把 stereo 当 mono 直接裁半。

## C26.2 — 帧长与包长一致（P0）

- `frame_samples = sample_rate_hz * frame_ms / 1000 * channels`。
- 语音/AEC/Opus 常用 10ms / 20ms；Opus frame duration 只能选 2.5/5/10/20/40/60ms。
- Queue payload、encoder 输入、DMA half-buffer 的样本数必须用同一公式推导，禁止 magic 512/1024。

## C26.3 — 视频像素格式与 stride 一致（P1）

- 每帧必须声明 width、height、pixel_format、stride_bytes。
- RGB565 stride 至少 `width * 2`；RGB888 至少 `width * 3`；YUV422 至少 `width * 2`。
- Camera 输出格式与 LCD/LVGL 输入格式不同，必须有显式转换或硬件 PXP/DMA2D 路径。

## C26.4 — 转换器有界且不在热路径分配（P1）

- resampler、channel mixer、colorspace converter、scaler 使用预分配 workspace。
- 禁止在 `encode_frame` / `decode_frame` / `convert_frame` / `process_pcm` 中 `malloc/free/printf/LOG_*`。
- 大 buffer 使用 pool/ring，按最大 frame size 预先申请。

## C26.5 — codec 生命周期独立于每帧处理（P0）

- encoder/decoder handle 在 start/open 阶段创建，在 stop/cleanup 阶段释放。
- 禁止每帧 `opus_encoder_create`、`codec_open`、`jpeg_decoder_init`。
- 格式变更必须 reset/flush codec，并丢弃旧 generation 的 stale output。

## C26.6 — 格式协商与遥测（P2）

至少保留：

- negotiated sample rate / channels / bit depth / pixel format
- format mismatch count
- codec error count
- max encode/decode time
- last frame size

## 推荐结构

```c
#define MEDIA_SAMPLE_RATE_HZ      16000U
#define MEDIA_CHANNELS            1U
#define MEDIA_BITS_PER_SAMPLE     16U
#define MEDIA_FRAME_MS            20U
#define MEDIA_FRAME_SAMPLES       (MEDIA_SAMPLE_RATE_HZ * MEDIA_FRAME_MS / 1000U)

typedef struct {
    uint16_t width;
    uint16_t height;
    uint16_t stride_bytes;
    media_pixel_format_t pixel_format;
} media_video_format_t;

static bool media_audio_format_match(const media_audio_format_t *a,
                                     const media_audio_format_t *b)
{
    return a->sample_rate_hz == b->sample_rate_hz &&
           a->channels == b->channels &&
           a->bits_per_sample == b->bits_per_sample;
}
```

## 反模式

```c
#define I2S_SAMPLE_RATE_HZ 48000
#define ASR_SAMPLE_RATE_HZ 16000
#define AUDIO_FRAME_MS 20
#define AUDIO_FRAME_SAMPLES 512

void encode_frame(int16_t *pcm)
{
    void *enc = opus_encoder_create(16000, 1, OPUS_APPLICATION_VOIP, NULL);
    uint8_t *tmp = malloc(4096);
    printf("encode\n");
    opus_encode(enc, pcm, 512, tmp, 4096);
}
```
